XML словари

Что такое XML словарь?

XML словарь (или XML схема) определяет структуру и содержание XML документа. Он задает правила, которым должен соответствовать XML документ.

Типы XML словарей:

DTD

Document Type Definition - самый старый способ определения структуры XML.

<!ELEMENT note (to, from, body)>
<!ELEMENT to (#PCDATA)>
<!ELEMENT from (#PCDATA)>
<!ELEMENT body (#PCDATA)>

XSD

XML Schema Definition - более мощная альтернатива DTD.

<xs:element name="note">
  <xs:complexType>
    <xs:sequence>
      <xs:element name="to" type="xs:string"/>
      <xs:element name="from" type="xs:string"/>
      <xs:element name="body" type="xs:string"/>
    </xs:sequence>
  </xs:complexType>
</xs:element>

RelaxNG

Более простой и удобный для чтения формат схемы.

<element name="note">
  <element name="to"><text/></element>
  <element name="from"><text/></element>
  <element name="body"><text/></element>
</element>

Создание собственного XML словаря

Чтобы создать собственный XML словарь, выполните следующие шаги:

  1. Определите структуру данных, которые вы хотите представить
  2. Создайте элементы и атрибуты, которые будут описывать ваши данные
  3. Определите отношения между элементами (вложенность, последовательность)
  4. Укажите типы данных для элементов и атрибутов
  5. Проверьте вашу схему на реальных данных

Пример создания простого словаря

Допустим, мы хотим создать словарь для хранения информации о книгах:

XSD схема (books.xsd)
<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema">
  <xs:element name="library">
    <xs:complexType>
      <xs:sequence>
        <xs:element name="book" maxOccurs="unbounded">
          <xs:complexType>
            <xs:sequence>
              <xs:element name="title" type="xs:string"/>
              <xs:element name="author" type="xs:string"/>
              <xs:element name="year" type="xs:integer"/>
              <xs:element name="price" type="xs:decimal"/>
            </xs:sequence>
            <xs:attribute name="id" type="xs:string" use="required"/>
          </xs:complexType>
        </xs:element>
      </xs:sequence>
    </xs:complexType>
  </xs:element>
</xs:schema>
XML документ (library.xml)
<?xml version="1.0"?>
<library xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:noNamespaceSchemaLocation="books.xsd">
  <book id="bk001">
    <title>XML для начинающих</title>
    <author>Иван Петров</author>
    <year>2022</year>
    <price>29.99</price>
  </book>
  <book id="bk002">
    <title>Продвинутый XML</title>
    <author>Мария Сидорова</author>
    <year>2023</year>
    <price>39.99</price>
  </book>
</library>
