Основы XML

Что такое XML?

XML (eXtensible Markup Language) - это язык разметки, который определяет набор правил для кодирования документов в формате, который является одновременно удобочитаемым и машиночитаемым.

Основные особенности XML:

  • XML предназначен для хранения и передачи данных
  • XML является расширяемым - вы можете создавать свои собственные теги
  • XML не зависит от платформы и может использоваться в различных системах
  • XML поддерживает Unicode, что позволяет использовать различные языки

Структура XML документа

Каждый XML документ должен содержать:

  1. Объявление XML (необязательно, но рекомендуется)
  2. Корневой элемент, который содержит все остальные элементы
  3. Правильно вложенные элементы

Пример правильного XML:

<?xml version="1.0" encoding="UTF-8"?>
<catalog>
  <book id="bk101">
    <author>Гамбардов</author>
    <title>XML в действии</title>
    <price>29.99</price>
  </book>
</catalog>

Пример неправильного XML:

<?xml version="1.0"?>
<catalog>
  <book id="bk101">
    <author>Гамбардов
      <title>XML в действии</author>
    </title>
    <price>29.99
</catalog>

Ошибки: незакрытые теги, неправильная вложенность

XML пространства имен

Пространства имен XML используются для избежания конфликтов имен элементов и атрибутов.

<root xmlns:h="http://www.w3.org/TR/html4/"
     xmlns:f="http://www.w3schools.com/furniture">
  
  <h:table>
    <h:tr>
      <h:td>Apples</h:td>
      <h:td>Bananas</h:td>
    </h:tr>
  </h:table>
  
  <f:table>
    <f:name>Coffee Table</f:name>
    <f:width>80</f:width>
  </f:table>
</root>
