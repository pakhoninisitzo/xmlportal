Быстрый старт с XML

XML (eXtensible Markup Language) - это язык разметки, предназначенный для хранения и передачи структурированных данных.

<?xml version="1.0" encoding="UTF-8"?>
<book>
  <title>XML для начинающих</title>
  <author>Иван Иванов</author>
  <year>2023</year>
</book>
